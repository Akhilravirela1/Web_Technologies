Virtual Library

Addendum

problem addressed

In the JavaScript code, the problem addressed was to enable the addition of books to the virtual library when the "Add Book" button is clicked.

Overview of the Solution

To achieve this, an event listener was implemented to capture the click event on the button and prevent the default form submission behavior using event.preventDefault(). The function addBook then extracts the title and author values from input fields, creates a new list item with the book details, and appends it to the book list. The input fields are then cleared for subsequent entries. A conditional alert was included to prompt the user to enter both title and author if either field is empty.

conclusion

The process involved straightforward DOM manipulation to dynamically add book entries based on user input. No significant difficulties were encountered, and the solution focused on ensuring a seamless addition of books to the library while providing user feedback in case of missing information.
